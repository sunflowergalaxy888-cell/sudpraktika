# Стаття 35. П

равові наслідки повторності, сукупності та рецидиву кримінальних правопорушень Повторність, сукупність та рецидив кримінальних правопорушень враховуються при кваліфікації кримінальних правопорушень та призначенні покарання, при вирішенні питання щодо можливості звільнення від кримінальної відповідальності та покарання у випадках, передбачених цим Кодексом. {
