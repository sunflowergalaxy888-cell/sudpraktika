# Стаття 356. С

амоправство Самоправство, тобто самовільне, всупереч установленому законом порядку, вчинення будь-яких дій, правомірність яких оспорюється окремим громадянином або підприємством, установою чи організацією, якщо такими діями була заподіяна значна шкода інтересам громадянина, державним чи громадським інтересам або інтересам власника, - карається штрафом до п'ятдесяти неоподатковуваних мінімумів доходів громадян або виправними роботами на строк до двох років, або пробаційним наглядом на строк до двох років. {
