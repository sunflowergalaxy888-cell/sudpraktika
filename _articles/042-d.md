# Стаття 42. Д

іяння, пов'язане з ризиком 1. Не є кримінальним правопорушенням діяння (дія або бездіяльність), яке заподіяло шкоду правоохоронюваним інтересам, якщо це діяння було вчинене в умовах виправданого ризику для досягнення значної суспільно корисної мети. 2. Ризик визнається виправданим, якщо мету, що була поставлена, не можна було досягти в даній обстановці дією (бездіяльністю), не поєднаною з ризиком, і особа, яка допустила ризик, обґрунтовано розраховувала, що вжиті нею заходи є достатніми для відвернення шкоди правоохоронюваним інтересам. 3. Ризик не визнається виправданим, якщо він завідомо створював загрозу для життя інших людей або загрозу екологічної катастрофи чи інших надзвичайних подій. {
