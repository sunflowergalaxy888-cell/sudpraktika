# Стаття 138. Н

езаконна лікувальна діяльність Заняття лікувальною діяльністю без спеціального дозволу, здійснюване особою, яка не має належної медичної освіти, якщо це спричинило тяжкі наслідки для хворого, - карається виправними роботами на строк до двох років, або обмеженням волі на строк до трьох років, або позбавленням волі на строк до трьох років.
