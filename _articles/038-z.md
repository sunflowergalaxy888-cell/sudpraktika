# Стаття 38. З

атримання особи, що вчинила кримінальне правопорушення 1. Не визнаються кримінально протиправними дії потерпілого та інших осіб безпосередньо після вчинення посягання, спрямовані на затримання особи, яка вчинила кримінальне правопорушення, і доставлення її відповідним органам влади, якщо при цьому не було допущено перевищення заходів, необхідних для затримання такої особи. 2. Перевищенням заходів, необхідних для затримання особи, яка вчинила кримінальне правопорушення, визнається умисне заподіяння особі, що вчинила кримінальне правопорушення, тяжкої шкоди, яка явно не відповідає небезпечності посягання або обстановці затримання особи, яка вчинила кримінальне правопорушення. Перевищення заходів, необхідних для затримання особи, яка вчинила кримінальне правопорушення, має наслідком відповідальність лише у випадках, спеціально передбачених у статтях 118 та 124 цього Кодексу. {
